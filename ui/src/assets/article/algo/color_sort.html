对三类不同元素进行排序

问题描述:

给定一个数组, 数组中只包含'S', 'M' 和 'L'。它们分别代表着小号、中号或大号三种尺寸。设计一个算法, 将它们由小到大排序，

例子:

    输入: ['L', 'S', 'L', 'M', 'M', 'S']
    输出: ['S', 'S', 'M', 'M', 'L', 'L']
  
    输入: ['L', 'S', 'M']
    输出: ['S', 'M', 'L']

分析:

这类问题是对三种不同类型的元素进行排序。这三种元素可以是 0, 1, 2。 可以是 'S', 'M', 'L'。可以是三种水果 'Apple', 'Banana', 'Kiwi'。解决问题的思路是一样的。我们可以用两 个指针的方法来解决这类问题。假设, 我们定义了两个指针 p1 和 p2。如果我们维持p1指针的 左边全部为第一类元素, p2指针的右边全部为第三类元素。这里, 我们定义第三个指针i用来从 头扫描数组的元素。那么, 当i指针和p2指针相遇之后, 任务就完成了。因为, p1 和 p2之间的元素一定是第二种元素。

    def sort_size(sizes):
