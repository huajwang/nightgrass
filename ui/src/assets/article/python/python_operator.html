Python Operators

Python Operators

Operators are used to perform operations on variables and values. Python divides the operators in the following groups:

  • Arithmetic operators: e.g., +, -, *, /
  • Assignment operators: =
  • Comparison operators: >, >=, <, <=, ==
  • Logical operators: and, or, not
  • Identity operators: is
  • Membership operators: in
  • Bitwise operators

算术运算符

Arithmetic operators are used with numeric values to perform common mathematical operations:

Operator Name Example
+ 加 x + y
- 减 x - y
* 乘 x * y
/ 除 7 / 2 得 3.5
// 取整除(Floor division) 7 // 2 得 3
% 取余数(Modulus) 7 % 2 得 1
** 乘方(Exponentiation) 2 ** 3 得 8

赋值操作符

Assignment operators are used to assign values to variables。下面列出部分的 操作符:

operator example same as
= x = 8 x = 8
+= x += 1 x = x + 1
-= x -= 1 x = x - 1

比较操作符

Comparison operators are used to compare two values:

operator name example
> 大于 x > y
>= 大于等于 x >= y
< 小于 x < y
<= 小于等于 x <= y
== 等于 x == y
!= 不等于 x != y

逻辑操作符

Logical operators are used to combine conditional statements:

operator explanation example
and Returns True if both statements are true x >= 80 and x < 90
or Returns True if one of the statements is true x < 0 or x > 100
not Reverse the boolean value, returns True if the value is False not (x >= 60)

Identity Operators

Identity operators are used to compare the objects, not if they are equal, but if they are actually the same object, with the same memory location:

operator explanation example
is Returns True if both variables are the same object x is y
is not Returns True if two variables are two different objects x is not y

== 与 is 的区别

计算机程序处理各种各样的数据。在我们的程序中, 定义变量(variables)来跟这些数据进行联系, 程序通过操控这些变量来实现对 相应数据的增删查改。在程序运行时, 这些数据被 loaded 到计算机的内存(memory)中, 我们可以通过变量来找到相应的数据。

在程序运行时, 变量会被分配计算机内存用于存储数据。我们可以调用 id() built-in 方法来 查看一个变量在内存中的地址: 

      nums1 = [1, 2, 3]
      nums2 = [1, 2, 3]
      print(id(nums1))
      print(id(nums2))
      print(nums1 == nums2)
      print(nums1 is nums2) # nums1, nums2 指向两个不同的数据对象

从下面的输出我们可以看到 nums1 和 nums2 指向不同的内存地址。也就是说 nums1 和 nums2 这两个变量指向不同的数据对象。而这两个对象的值是一样的, 都是 [1, 2, 3]:

      18992520
      56398184
      True
      False

再请看下面的例子:

      nums1 = [1, 2, 3]
      nums2 = nums1
      print(id(nums1))
      print(id(nums2))
      print(nums1 == nums2)
      print(nums1 is nums2) # nums1, nums2 指向同一个数据对象

运行上面的程序得到如下所示输出:

      9424264
      9424264
      True
      True

从上面的例子可以看出, 只有当两个变量(比如 nums1, nums2)指向同一数据对象(内存地址)时, nums1 is nums2 的计算结果才为 True, 否则为 False; 但是, 只要两个变量指向的数据对象的值相等, 那么 nums == nums 就为 True。
