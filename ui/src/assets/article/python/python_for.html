Python For 循环

for loop

for 循环用于 iterate/loop through 集合序列（比如, 列表、tuple、字典、set 或字符串）中的数据。 使用 for 循环，我们可以对列表、tuple、set 等数据集合中的每个数据进行访问: 

      fruits = ["apple", "banana", "kiwi"]
      for fruit in fruits:
          print(fruit)

和 while 循环不一样的是, 我们不需要为 for 循环事先初始化一个循环变量 (loop variable 或者称为 indexing variable)。一般来说, while 循环是用一个循环变量 来控制循环的执行的。

上面的 for 循环很容易让我们遍历一个列表的每一个数据项。但在有些情况下, 我们不仅需要数据项的 值, 也需要数据项的 index 位置, 那么我们可以使用 Python 提供的 enumerate() built-in 方法 来实现: 

      fruits = ["apple", "banana", "kiwi"]
      for idx, fruit in enumerate(fruits):
          print(f"第 {idx + 1} 个元素为 { fruit }")

looping through 字符串

字符串包含着一系列字符, 是可 iterable 的对象:

      s = "How do you do?"
      for c in s:
          print(c)

break statement

和 while 循环一样, 我们可以使用 break 来终止 for 循环的执行。比如: 

      fruits = ["apple", "banana", "kiwi", "cherry", "grape", "melon"]
      for f in fruits:
          if f == "kiwi":
              print("Kiwi Found! Exit for loop.")
              break # 程序退出 for 循环
          print(f"The current fruit is { f }")

With the break statement we can stop the loop before it has looped through all the items。

continue statement

With the continue statement we can stop the current iteration of the loop, and continue with the next:

      fruits = ["apple", "banana", "kiwi", "cherry", "grape", "melon"]
      for f in fruits:
          if f == "kiwi":
              continue
          print(f)

在上面的程序片段中, 除了 'kiwi' 之外, 所有水果都会被打印出来。

Else in For Loop

The else keyword in a for loop specifies a block of code to be executed when the loop is finished:

      fruits = ["apple", "banana", "kiwi"]
      for f in fruits:
          print(f)
      else:
          print("For loop 执行完毕!")
      print("after for loop")

如果在 for 循环体中, 程序执行了 break 程序语句, 那么 else 程序块将不会被执行。比如: 

      fruits = ["apple", "banana", "kiwi", "cherry"]
      for f in fruits:
          print(f)
          if f == "banana":
              break # 这个语句执行以后, 程序退出循环, else 程序块不会被执行
      else:
          print("For loop 执行完毕!")
      print("after for loop")

Nested Loops

A nested loop is a loop inside a loop. The "inner loop" will be executed one time for each iteration of the "outer loop": 

      fruits = ["apple", "banana", "cherry"]
      sizes = ["small", "medium", "large"]
      for fruit in fruits: # 外循环
          for size in sizes: # 内循环
              print(fruit + " : " + size)

在上面的程序片段中, 每一种水果都会执行内循环。输出如下所示:


      apple : small
      apple : medium
      apple : large
      banana : small
      banana : medium
      banana : large
      cherry : small
      cherry : medium
      cherry : large

The range() Function

To loop through a set of code a specified number of times, we can use the range() function, The range() function returns a sequence of numbers, starting from 0 by default, and increments by 1 (by default), and ends at a specified number. 

      for n in range(5):
          print(n)

输出如下所示:


      0
      1
      2
      3
      4

Note that range(5) is not the values of 0 to 5, but the values 0 to 4.

The range() function defaults to 0 as a starting value, however it is possible to specify the starting value by adding a parameter: range(2, 6), which means values from 2 to 6 (but not including 6): 

      for n in range(2, 6):
          print(n)

The range() function defaults to increment the sequence by 1, however it is possible to specify the increment value by adding a third parameter range(2, 20, 5): 

      for n in range(2, 20, 5):
          print(n)

输出如下所示:


      2
      7
      12
      17
