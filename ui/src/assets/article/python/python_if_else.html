Python 分支逻辑判断

Python Conditions and If statements

Python 支持如下的比较逻辑运算, 用于比较两个值的大小:

  • a == b
  • a != b
  • a < b
  • a <= b
  • a > b
  • a >= b
    • 其中 a, b 是两个 Python 变量

在 Python 的逻辑运算中, 我们还经常使用到如下的逻辑运算:

  • and
  • or
  • not

These conditions can be used in several ways, most commonly in "if statements" and loops. 今天我们先介绍 if statements.

An "if statement" is written by using the if keyword: 

      x = 18
      y = 15
      if x > y:
          print(f"{ x } > { y }")

在上面这个例子中，我们使用两个变量 x 和 y, 它们被用作 if 语句的一部分来测试 x 是否大于 y。 因为 x 是 18, y 是 15, 我们知道 18 大于 15，所以屏幕上会打印 18 > 15。

elif

The elif keyword 表示 "if the previous conditions were not true, then try this condition". 在 if 语句的后面, 我们可以用 elif 语句来检查另外的条件是否满足: 

      x = 18
      y = 18
      if x > y: # the first condition
          print(f"{ x } > { y }")
      elif x == y: # the seocnd condition
          print(f"{ x } == { y }")

如果 x > y, 那么就运行 print(f"{ x } > { y }")。否则, 继续检查 x == y 是否成立, 如果成立程序就会执行 print(f"x == y")

else

The else keyword catches anything which isn't caught by the preceding conditions. 

      x = 15
      y = 18
      if x > y: # the first condition
          print(f"{ x } > { y }")
      elif x == y: # the seocnd condition
          print(f"{ x } == { y }")
      else: # 如果前面的条件都不成立, 那么程序就会执行 else 的部分
          print(f"{x} < {y}")

In this example the first condition is not true, also the elif condition is not true, so the program goes to the else condition and execute print(f"{x} < {y}") statement. You can also have an else without the elif: 

      x = 15
      y = 15
      if x > y:
          print(f"{x} > {y}")
      else: # 除非满足第一个条件, 不然所有其它的情况都执行 else 部分
          print(f"{x} <= {y}")

如果程序只需要分别处理两种不同情况, 那么可以使用下面的简短形式:

      x = 15
      y = 15
      print(f"{x} > {y}") if x > y else print(f"{x} <= {y}")

and 逻辑运算

The and keyword is a logical operator, and is used to combine conditional statements:

      x = 3
      y = 4
      z = 5
      if z > x and z > y:
          print("Both conditions are True")

在上面的例子中, 只有当 z > x 和 z > y两个条件都满足的时候, print("Both conditions are True") 才会被执行。

or 逻辑运算

The or keyword is a logical operator, and is used to combine conditional statements:

      x = 8
      y = 4
      z = 5
      if z > x or z > y:
          print("At lieast one condition is True")

只要 z > x 或者 z > y 中有一个条件满足, print("At lieast one condition is True") 语句就会被执行。

not 逻辑运算

not 逻辑运算符将 True 变成 False, 把 False 变成 True。例如: 

      if not 5 > 6:
          print("5 <= 6")

上面的 if 条件判断语句, 先计算5 > 6 的值, 其 运算结果为 False, 再经过 not False 运算, 最终结果为 True。 所以, print("5 <= 6") 会被运行。 从上面的例子中, if not 5 > 6 语句中涉及到了两种不同的运算符号: not 和 > 。 下一节课, 我们将会介绍运算符号的计算优先级问题。

nested if

You can have if statements inside if statements, this is called nested if statements. 

      mark = int(input("Enter your mark: "))
      if mark >= 60: # outter if
          if mark > 90: # nested if
              print("Excellent!")
          else:
              print("Pass")
      else:
          print("Fail!")
