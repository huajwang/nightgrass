增添数据到列表中

使用列表的 append() 方法, 将数据项添加到列表的末尾:

      fruits = ["apple", "banana", "cherry"]
      fruits.append("kiwi")
      print(fruits)

append() 方法将新的数据添加到列表的末尾。要想把数据添加到列表的特定位置, 我们应该使用 insert() 方法: 

      fruits = ["apple", "banana", "cherry"]
      # List 的 insert() 方法有两个参数, 第一个参数代表新数据的加入位置( index )。
      fruits.insert(1, "blackberry")

从列表中删除数据

List 提供了几种不同的方法, 从列表中删除数据。调用 remove() 方法从列表中 删除指定的数据项: 

      fruits = ["apple", "banana", 'orange', "cherry"]
      fruits.remove("orange")
      print(fruits)

调用 List 的 pop() 方法来删除数据:

      fruits = ["apple", "banana", 'orange', "cherry"]
      # 从列表中删除 index 为 1 的数据
      f = fruits.pop(1)
      print(f"The deleted fruit is { f }")
      print(f"The current fruits = { fruits }")

在调用 pop() 方法时, 如果你没有指定 index 参数, 那么列表的最后一个数据项将会被删除: 

      fruits = ["apple", "banana", 'orange', "cherry"]
      # pop() 方法里没有指定 index
      f = fruits.pop()
      print(f"The deleted fruit is { f }")
      print(f"The current fruits = { fruits }")

除了 remove(), pop() 方法之外, 我们还可以用 del 关键字 来删除指定位置的元素: 

      fruits = ["apple", "banana", "kiwi"]
      del fruits[0]
      print(fruits)

你如果要把整个列表的数据全部删掉, 那么就使用 clear() 方法, 或者 del 关键字: 

      fruits = ["apple", "banana", "kiwi"]
      fruits.clear()
      print(fruits) # fruits 现在为空列表 []
    
      fruits = ["apple", "banana", "kiwi"]
      del fruits
      print(fruits) # 会出现 NameError: name 'fruits' is not defined

clear() 和 del 的差别在于, clear() 只是把列表中的数据清空。 而 del 关键字 则是把整个列表变量完全删除。

Copy a List

假设我们已经定义了一个列表 list1 , 现在你要把 list1 的内容复制到另一个 列表 list2 。如果你使用 list2 = list1 来复制列表, 那么, list1 和 list2 两个变量都指向同一个列表。而在 list1 中所做的更改也将自动影响到 list2 , 反之亦然, 因为 list1 和 list2 都指向同一个列表。请看下面的代码: 

      list1 = [1, 2, 3, 4, 5]
      list2 = list1
      # is 关键字用来判断两个变量是否指向同一 object
      print(list1 is list2) # True
      print(list2 is list1) # True
      list1[0] = 6
      print(list1) # [6, 2, 3, 4, 5]
      print(list2) # [6, 2, 3, 4, 5]

你可以使用列表的 built-in 方法 copy() : 

      list1 = [1, 2, 3, 4, 5]
      # 调用 List 的 copy() 方法。list1 和 list2 指向两个不同的列表
      list2 = list1.copy()
      print(list1 is list2) # False
      print(list2 is list1) # False
      print(list1 == list2) # True。因为它们的值都是 [1, 2, 3, 4, 5]
      list1[0] = 6
      print(list1) # [6, 2, 3, 4, 5]
      print(list2) # list2 不受影响, 仍然是 [1, 2, 3, 4, 5]

除了使用 List 的 built-in copy() 之外, 你也可以使用 Python 提供的 list() 方法来达到相同的目的: 

      list1 = [1, 2, 3]
      list2 = list(list1) # list1 和 list2 指向两个不同的列表, 虽然两个列表的值都为 [1, 2, 3]
      print(list1 is list2) # False
      print(list2 is list1) # False

Join Two Lists

在 Python 中，有几种方法可以连接(join)两个或多个列表。最简单的方法就是使用 + 运算符: 

      list1 = ["a", "b" , "c"]
      list2 = [1, 2, 3]
      list3 = list1 + list2
      print(list3)

连接两个列表的另一种方法是将列表2中的所有项逐个附加到列表1中： 

      list1 = ["a", "b" , "c"]
      list2 = [1, 2, 3]
      # 使用 append() 方法将 list2 中的数据一个个加入到 list1
      for x in list2:
          list1.append(x)
      print(list1)

或者，使用 extend() 方法，将元素从一个列表添加到另一个列表:

      list1 = ["a", "b" , "c"]
      list2 = [1, 2, 3]
      list1.extend(list2)
      print(list1)
