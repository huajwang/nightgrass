Python 变量

创建 Python Variables

Variables are containers for storing data values. Unlike other programming languages, Python has no command for declaring a variable. A variable is created the moment you first assign a value to it. 

      name = "John Winters"
      age = 30
      gender = "M"

Variables do not need to be declared with any particular type and can even change type after they have been set. 

      x = 4 # x is of type int
      x = "Grace" # x is now of type str
      print(x)

String variables can be declared either by using single or double quotes: 

      s1 = "The Great Wall"
      s2 = 'The Great Wall'

Variable Names

A variable can have a short name (like x and y) or a more descriptive name (age, carname, total_volume). Rules for Python variables:

  • A variable name must start with a letter or the underscore character;
  • A variable name cannot start with a number;
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • Variable names are case-sensitive (age, Age and AGE are three different variables)

下面为合法的变量名:

      dogname = "stone"
      dog_name = "stone"
      _dog_name = "stone"
      dogName = "stone"
      DOGNAME = "stone"
      dogname1 = "stone"

下面所列的变量名是非法的:

      3dogname = "stone"
      dog-name = "stone"
      dog name = "stone"

Assign Value to Multiple Variables

Python allows you to assign values to multiple variables in one line: 

      x, y, z = 3, 4, 5
      f1, f2 = "apple", "cherry"

And you can assign the same value to multiple variables in one line: 

      x = y = z = "Apple"

Output Variables

The Python print statement is often used to output the value of a variable. To combine both text and a variable, Python uses the + character: 

      age = 12
      print("I am " + age)

我们可以使用 Python 的 F-strings 机制来格式化字符串文本。要创建一个 F-string, 只要在字符串前面加上字母 f 即可。F-strings 提供了一种简洁的方法, 让我们可以将 Python 表达式嵌入字符串文本中进行格式化。比如: 

      name = "Michael Waters"
      age = 12
      s = f"My name is {name}. I am {age} years old."
      print(s)
      print(f"Hello {name}. Nice to meet you.")

运行上面的程序片段, 你会看到如下输出:


  My name is Michael Waters. I am 12 years old.
  Hello Michael Waters. Nice to meet you.
