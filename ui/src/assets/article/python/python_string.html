Python 字符串

字符串常量

Python 字符串由单引号或双引号包围。比如: 

      s1 = 'How is everything?'
      s2 = "How's everything?"
      s3 = 'My favorite movie is "Lion King"'
      print(s1)
      print(s2)
      print(s3)
    
      How is everything?
      How's everything?
      My favorite movie is "Lion King"

如果字符串里面要包含单引号, 那么字符串就用双引号括起来; 如果字符串里要包含双引号, 那么字符串就用单引号括起来。

Multiline Strings

You can assign a multiline string to a variable by using three double quotes:

      s = """A woman gets on a bus with her baby. The driver says "Ugh – that's the ugliest baby I've ever seen!"
      The woman walks to the back of the bus and sits down.
      She says to the man next to her: “The driver just insulted me!"
      The man says: "You go up there and tell him off. Go on. I’ll hold your monkey for you." """

      print(s)

Or three single quotes: 

      s = '''A woman gets on a bus with her baby. The driver says "Ugh – that's the ugliest baby I’ve ever seen!"
      The woman walks to the back of the bus and sits down.
      She says to the man next to her: "The driver just insulted me!"
      The man says: “You go up there and tell him off. Go on. I’ll hold your monkey for you."'''

      print(s)

Strings are Arrays

Like many other popular programming languages, strings in Python are arrays of bytes representing unicode characters. However, Python does not have a character data type, a single character is simply a string with a length of 1. Square brackets can be used to access elements of the string.

Get the character at position 1 (remember that the first character has the position 0): 

      s = "Hello, World!"
      print(s[1])

slicing

You can return a range of characters by using the slice syntax. Specify the start index and the end index, separated by a colon, to return a part of the string. E.g. get the characters from position 2 to position 5 (not included): 

      s = "Hello, World!"
      print(s[2:5])

Negative Indexing

Use negative indexes to start the slice from the end of the string. E.g. Get the characters from position 5 to position 1, starting the count from the end of the string: 

      s = "Hello, World!"
      print(s[-5:-2])

You can mix up positive and negative indexes to slice a string. You can also skip the start or the end index of the string: 

      s = "Hello, World!"
      print(s[-5:])
      print(s[-5:12])
      print(s[:-5])
      print(s[2:-6])

String Length

To get the length of a string, use the len() function: 

      s = "Hello, World!"
      print(len(s))

String methods

Python has a set of built-in methods that you can use on strings.

The split() method splits the string into substrings if it finds instances of the separator: 

      s = "Once upon a time, there is a little cabin in the woods."
      words = s.split()
      print(words)
    
    ['Once', 'upon', 'a', 'time,', 'there', 'is', 'a', 'little', 'cabin', 'in', 'the', 'woods.']

By default, split() splits the string into substrings by white spaces. You can specify the separator by providing it as parameter of the method. e.g. 

      s = "Once upon  a time, there is a little cabin in the woods."
      words = s.split(',') # specify , as separator
      print(words)
    
    ['Once upon  a time', ' there is a little cabin in the woods.']

The strip() method removes any whitespace from the beginning or the end: 

      s = "     Hello World!  "
      s1 = s.strip()
      print(s1)

The replace() method replaces a string with another string: 

      s = "Once upon a time, there is a little cabin in the woods."
      s1 = s.replace('cabin', 'house')
      print(s)
      print(s1)
    
    Once upon a time, there is a little cabin in the woods.
    Once upon a time, there is a little house in the woods.

Note: The original string s remains unchanged. The call to replace() method returns a new string.

Check String

To check if a certain phrase or character is present in a string, we can use the keywords in or not in: 

      s = "Once upon a time, there is a little cabin in the woods."
      if 'cabin' in s: # in
          print('yes')
      else:
          print('no')


      if 'house' not in s: # not in
          print('yes')
      else:
          print('no')

String Concatenation

To concatenate, or combine, two strings you can use the + operator: 

      s1 = 'back to'
      s2 = 'the future'
      s3 = s1 + " " + s2
      print(s1)
      print(s2)
      print(s3)

We cannot combine strings and numbers like this: 

      s1 = "I am "
      age = 12
      s = s1 + age # TypeError: can only concatenate str (not "int") to str

You can use f-strings to concatenate strings and numbers: 

      s1 = "I am"
      age = 12
      s = f"{s1} {age}"
      print(s)

Or we can combine strings and numbers by using the format() method. The format() method takes the passed arguments, formats them, and places them in the string where the placeholders {}are: 

      s1 = "My name is Michael, and I am {}"
      age = 12
      s2 = s1.format(age) # call format() on s1 and pass in age as parameter
      print(s1)
      print(s2)

The format() method can take multiple numbers of arguments, and are placed into the respective placeholders: 

      name = 'Michael'
      age = 12
      town = 'Toronto'
      s1 = "My name is {}. I am {}. I am from {}."
      s2 = s1.format(name, age, town)
      print(s1)
      print(s2)
    
    My name is {}. I am {}. I am from {}.
    My name is Michael. I am 12. I am from Toronto.

You can use index numbers {0} to be sure the arguments are placed in the correct placeholders: 

      name = 'Michael'
      age = 12
      town = 'Toronto'
      s1 = "My name is {1}. I am {2}. I am from {0}."
      s2 = s1.format(town, name, age)
      print(s1)
      print(s2)

Escape Character

To insert characters that are illegal in a string, use an escape character. An escape character is a backslash \ followed by the character you want to insert. An example of an illegal character is a double quote inside a string that is surrounded by double quotes: 

      s = "My favorite movie is \"Lion King\"."
      print(s)

For the above example, you can use a single quote instead of using \: 

      s = 'My favorite movie is "Lion King".'
      print(s)

There are some other escape characters used in Python. e.g.: 

      s = "Welcome!\nPython is awesome!" # \n
      print(s)
    
    Welcome!
    Python is awesome!
