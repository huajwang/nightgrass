Python While 循环

Python 循环

Python 提供两种基本的循环:

  • while loop
  • for loop

while loop

With the while loop we can execute a set of statements as long as a condition is true. 例如, 使用下面的 while 循环来打印 0 - 4 的数字:

      i = 0 # 定义 loop variable i
      while i < 5: # while 的循环条件
          print(f"the value of i is { i }")
          i += 1   # update the value of loop variable i
      # program statements after while loop
      print('after while loop')

while 循环有三个基本的要素:

  • 循环变量(loop variable);
  • while 循环条件;
  • 在 while 循环体内修改循环变量值的语句;

在使用 while 循环时, 我们一般要先初始化一个循环变量(i = 0)。 对这个循环变量进行某种逻辑运算(循环条件 i < 5)。如果循环条件成立的话, 程序可以 进入到 while 循环体内运行。如果条件不成立的话, 程序不会进入 while 循环体内, 而是跳过 while 循环去执行 while 循环之后的程序。

在循环体内一般应该有修改 loop variable i 的程序语句( i + = 1 ), 否则, 循环会永远执行下去。

break statement

With the break statement we can stop the loop even if the while condition is true:

      i = 1
      while i < 6:
          print(f"i = { i }")
          if i == 3:
              break
          i += 1
      print(f"after while loop, i = { i }")

continue statement

With the continue statement we can stop the current iteration, and continue with the next:

      i = 0
      while i < 6:
          i += 1
          if i == 3:
              continue
          print(i)

在上面的程序片段中, 当 i 的值等于 3 时, continue 语句被执行, 那么当前循环的剩余语句, print(i) 就被略过, 而直接去执行下个循环。

else statement

With the else statement we can run a block of code once when the condition no longer is true: 

      i = 1
      while i < 6:
          print(i)
          i += 1
      else: # while loop 可以有 else 语句, 在循环条件不(再)满足时执行。
          print(f"此时, i = { i }, i < 6 的循环条件不再满足。")

注意: while 的 else 代码块只有在循环"正常"结束才会被执行。 如果循环是因为执行了 break 语句而退出的, 那么 else 里的程序语句是不会被执行的。请 看下面的例子: 

      i = 0
      while i < 6:
          print(i)
          if i == 3:
              break # i 等于 3 时, 退出循环
          i += 1
      else:
          print(f"i = { i }") # 这个语句不会被执行
