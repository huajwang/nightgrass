狗粮

问题描述:

旺财喜欢吃东西。在一天结束时，他要么高兴, 要么伤心，这取决于他这一天收到的狗粮的多少。这些狗粮有三种尺寸： 小号 S、中号M 和大号 L。 旺财的幸福指数可以用以下公式来衡量：
1 × S + 2 × M + 3 × L
其中, S是小食品的数量，M是中食品的数量, L是大餐的数量。如果旺财的幸福分数是 10 或更高, 那么他是幸福的。 否则，他很伤心。编写一个程序, 根据狗粮的数量, 来判断旺财在一天结束时是快乐还是悲伤。

例子:

    输入: 3, 1, 0
    输出: sad
  
    输入: 3, 2, 1
    输出: happy

输入的是小于 10 的三个非负整数, 分别代表旺财一天中吃到的小食品的数量 S，中食品的数量 M，和 大食物的数量 L。如果旺财的幸福指数等于或大于 10, 那么程序输出 happy。否则, 输出 sad。

算法:

    def ishappy(s, m, l):
        if s * 1 + m * 2 + l * 3 >= 10:
            return 'happy'
        else:
            return 'sad'
