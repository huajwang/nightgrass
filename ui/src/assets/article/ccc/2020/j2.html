流行病学

问题描述:

研究流行病学的人经常用模型来分析疾病的传播。在这个问题中，我们使用一个简单的模型:
当一个人感染了病毒, 他会在第二天感染另外 R 个人。一个人最多只会被感染一次。 设计一个算法来确定什么时候, 会有超过 P 个人感染了这种病毒。

例如:

    输入: 750, 1, 5
    输出: 4
  
    输入: 10, 2, 1
    输出: 5

其中, 输入为三个正整数。第一个整数 P 代表感染的总人数。第二个整数 X 表示刚开始(day 0) 有多少人感染了病毒。第三个整数 R 代表传染系数。

算法:

    def totalinfected(p, x, r):
      """
      p: 代表感染的总人数
      x: 初始被感染的人数
      r: 病毒的传播系数
      """
      # 定义列表 inf 来存储每一天的总感染人数。第0天有x个人感染。
      inf = [x]
      # d 代表第几天
      d = 0
      while sum(inf) < p:
          d += 1
          # 明天感染的总人数为今天感染人数 inf[-1] 的 r 倍。
          inf.append(inf[-1] * r)
      return d

上面我们使用了 dynamic programming 的方法, 利用当天的感染人数 * r 来得到下一天的感染人数。
