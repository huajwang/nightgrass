绘画艺术

问题描述:

小红一直在尝试一种新的艺术风格。她站在画布前，用画笔在画布上轻弹颜料滴。当她认为自己已经创造了一个杰作， 她用她的3D打印机打印一个框架来包围帆布。你的工作是帮助小红确定最小的矩形框的坐标，使每一滴油漆都位于框内。 框架上的点不在框架内。

例如:

    输入: [[44, 62], [34, 69], [24, 78], [42, 44], [64, 10]]
    输出: [23,9] [65,79]

输入输出说明:

  • 开括号必须用同一类型的括号关闭。
  • 打开的括号必须按正确的顺序闭合。
  • 空字符串也被认为是合法的。

例子:

    输入: "()"
    输出: true
  
    输入: "()[]{}"
    输出: true
  
    输入: "(}"
    输出: false
  
    输入: "([]{})"
    输出: true
  
    输入: "([{}])"
    输出: true
